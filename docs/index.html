Move your cursor.
With your hand, not your mouse.

Gesconi tracks your hand through a webcam and translates gestures into cursor movement, clicks, drags, and scrolls — no extra hardware, just Python and a camera.

02 · gesture map

Five gestures, five actions.

Each gesture is read from the relative position of your fingertips and knuckles — no calibration step required. This is the order Gesconi checks them in.

01

Open palm, index extended

Moves the cursor. The tip of your index finger maps directly to screen position, smoothed to reduce jitter.

move
02

Thumb + index pinch

A quick pinch-and-release fires a left click. The distance between thumb and index tips is measured every frame.

left click
03

Thumb + middle pinch

The same pinch gesture, one finger over, triggers a right click for context menus.

right click
04

Pinch and hold

Holding the thumb-index pinch past a short threshold switches into drag mode until you release.

drag
05

Index + middle raised together

Moving two fingers up or down scrolls the active window, similar to a two-finger trackpad swipe.

scroll
06

Closed fist

A closed fist pauses tracking entirely — useful for repositioning your hand without moving the cursor.

pause
03 · install

Run it on your own machine.

Gesconi is a Python script — clone it, install the three dependencies, and run it. A webcam is all you need.

terminal — gesconi setup
1
Clone the repository
git clone https://github.com/YOUR_USERNAME/gesconi.git
2
Move into the project and create a virtual environment
cd gesconi && python -m venv venv
3
Activate it (Windows shown — use source venv/bin/activate on macOS/Linux)
venv\Scripts\activate
4
Install dependencies
pip install -r requirements.txt
5
Run it
python gesconi.py
Before you run it: Gesconi has been tested with Python 3.11 and mediapipe==0.10.14 — newer Python versions can hit MediaPipe compatibility issues, so the virtual environment step above matters.
