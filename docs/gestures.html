+ GESTURE DICTIONARY

One hand, eight controls.

Each landmark relationship — the distance and position between fingertips — is translated into a distinct system-level action. Customize settings inside config files.

👆
01

Point

Index finger extended, others curled. Drives smooth, responsive cursor movement mapped directly to fingertip positions.

→ MOVE CURSOR
🤏
02

Single Pinch

Thumb and index fingertip pinch together quickly. Simulates a standard single left click event on release.

→ LEFT CLICK
⚡
03

Double Pinch

Two quick thumb-and-index fingertip pinches within 0.4 seconds triggers a double click action.

→ DOUBLE CLICK
👉
04

Right Click

Pinch thumb and middle finger together. Simulates a standard right click for triggers like context menus.

→ RIGHT CLICK
🎯
05

Drag & Drop

Pinch thumb and index and hold for 0.35s. Keeps the left mouse button held down, dragging items as your hand moves.

→ DRAG & DROP
↕️
06

Scroll Mode

Hold index, middle, ring, and pinky fingers up. Moving your palm up/down scrolls the active window relative to the center.

→ SCROLL WINDOW
📷
07

Screenshot

Pinch thumb and pinky finger together. Automatically captures a screenshot of your screen and saves it locally.

→ SCREENSHOT
⌨️
08

Type Letter

Make a closed fist and release. Releasing index types 'a', index+middle types 'b', and index+middle+ring types 'c'.

→ TYPE KEY
