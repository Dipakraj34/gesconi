+ INSTALL

Run it on your own machine.

Gesconi is a Python script — clone it, install the dependencies, and run it. A webcam is all you need.

terminal — gesconi setup
1
Clone the repository
git clone https://github.com/YOUR_USERNAME/gesconi.git
2
Move into the project directory and create a virtual environment
cd gesconi && python -m venv venv
3
Activate virtual environment (Windows shown — use source venv/bin/activate on macOS/Linux)
venv\Scripts\activate
4
Install packages dependencies
pip install -r requirements.txt
5
Run the Web App or Desktop tool
python gesconi.py # Web Dashboard
python gesconi.py --desktop # Original Desktop Tool
Python Requirements Note: Gesconi has been tested with Python 3.11 and mediapipe==0.10.14 — newer Python versions can hit MediaPipe compiler compatibility issues, so the virtual environment step above matters.
OpenCV — camera frame capturing
MediaPipe — hand landmarks processing
PyAutoGUI — system cursor triggers
Flask — local web dashboard engine
+ SYSTEM BENEFITS

Designed for speed & accessibility.

Gesconi redefines desktop mouse interaction without the need for expensive or bulky tracking gear.

♿

Accessibility

Provides an ergonomic, hands-free computer control option for users with physical mobility constraints.

⚡

Ultra Low Latency

MediaPipe compiles optimized WebAssembly and C++ threads to track hand joint movements in milliseconds.

⚙️

Customizable

Easily adjust threshold limits, map custom fingers configurations, or assign new hotkeys inside local config files.

🎥

Zero Extras

Works on any standard computer webcam — no specialized hardware, infrared cameras, or ring markers required.
